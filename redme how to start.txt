Для запуска тестов, выполните следующие шаги:

## Предварительные требования

1. Установите [PyCharm](https://www.jetbrains.com/pycharm/download/) на вашем компьютере.
2. Убедитесь, что Python версии 3.x установлен на вашем компьютере.

## Настройка проекта

1. Склонируйте репозиторий с тестами на свой компьютер.
2. Откройте проект в PyCharm.

## Запуск тестов

1. В PyCharm откройте файл, содержащий тесты (например, `test_a_post_api_key.py`).
2. Выберите тест, который вы хотите запустить (например, test_b_get_api_Pet_Category_List.py).
3. Щелкните правой кнопкой мыши на выбранном тесте и выберите "Run" (Запустить).

Продолжайте запускать остальные тесты поочередно по их распределению.

## Обработка возможных ошибок

1. При выполнении теста, который требует ID категории, убедитесь, что выбранный ID существует в списке доступных категорий. Вы можете использовать статическое значение категории, которое уже существует, чтобы избежать ошибок. Если необходимо, вы также можете создать категорию перед запуском теста для получения нового ID.
2. При редактировании категории, убедитесь, что id категория, которую вы пытаетесь отредактировать, уже существует. Вы также можете выбрать одну из ранее полученных категорий из списка.
3. При удалении созданной категории, убедитесь, что категория была успешно создана и ее ID известен.

Примечание: Указанные инструкции предполагают, что вы используете PyCharm для запуска тестов. Если вы используете другую IDE или консольный инструмент, пожалуйста, адаптируйте инструкции соответственно.

1. При выполнении теста, который требует ID животного, убедитесь, что выбранный ID существует в списке доступных животных. Вы можете использовать статическое значение категории, которое уже существует, чтобы избежать ошибок. Если необходимо, вы также можете создать животного перед запуском теста для получения нового ID.
2. При редактировании животного, убедитесь, что id животного, которую вы пытаетесь отредактировать, уже существует. Вы также можете выбрать одну из ранее полученных категорий из списка.
3. При удалении созданного животного, убедитесь, что животное было успешно создана и его ID известен.

Примечание: Указанные инструкции предполагают, что вы используете PyCharm для запуска тестов. Если вы используете другую IDE или консольный инструмент, пожалуйста, адаптируйте инструкции соответственно.
